# GrandReserva Client SDK

## Overview

Provides a Typescript library for integration with the GrandReserva API for
COP projects.

## Pre-requisites

- [NodeJS v18](https://nodejs.org/en)

## Local Setup

1. Install dependencies
   ```sh
   npm install
   ```

## Running Tests

1. Run `jest` unit tests
   ```sh
   npm run test
   ```
## Packaging the Library

1. Run build
   ```sh
   npm run build
   ```
